🔒 WHAT IS HASHING?

Hashing converts data into a fixed-length string. It's a one-way function — you cannot reverse a hash back to the original text. Used for password storage, file integrity, and digital signatures.

⚡ WHICH ALGORITHM?

MD5 is fast but insecure. SHA-1 has known weaknesses. SHA-256 is widely trusted and recommended. SHA-512 offers even stronger security for critical applications.
